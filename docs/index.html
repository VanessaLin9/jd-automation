Quick Summary

你可以先記住這四件事

The four most important things to know

1. 只在你主動使用時讀取頁面

1. It reads the page only when you use it

JD Saver 只有在你打開 popup 或按下儲存時，才會讀取目前那一頁的 JD 內容。

JD Saver reads the current job page only when you open the popup or choose to save that page.

2. 資料是寫進你自己的 Google Sheet

2. Your data goes to your own Google Sheet

extension 不會把職缺資料送到開發者自己的伺服器，主要寫入位置是你授權的 Google Sheet。

The extension does not send job data to a developer-owned server. It writes to the Google Sheet you authorize.

3. 只儲存少量設定資料

3. It stores only limited settings

本地只會保留語言、已連結表單清單、目前使用中的表單等必要設定，讓 extension 可以正常運作。

It stores only the settings needed to work properly, such as language, saved sheets, and your active sheet.

4. 你可以隨時停止使用或移除

4. You stay in control

你可以隨時中斷 Google 連線、刪除自己的 Google Sheet 紀錄，或直接解除安裝 extension。

You can disconnect Google, delete records from your sheet, or uninstall the extension at any time.

完整政策

Full Policy

正式版本

Formal policy text

JD Saver for Google Sheets 是一個 Chrome extension，幫助使用者把目前瀏覽中的職缺頁面存到使用者自己的 Google Sheet。

本 extension 可能處理的資料包括目前職缺頁中的欄位資訊、Google Sheet 的中繼資料與必要內容，以及 extension 正常運作所需的本地設定。

資料只會用於擷取職缺資訊、建立與管理使用者自己的 Google Sheet、檢查重複資料，以及將資料寫入使用者選定的表單。

extension 不會把職缺資料傳到開發者自有伺服器，不會販售使用者資料，也不會將資料用於廣告、使用者側寫或資料轉售。

使用者可以隨時中斷 Google 授權、刪除自己 Google Sheet 中的紀錄，或移除 extension。

JD Saver for Google Sheets is a Chrome extension that helps users save the current job description page to a Google Sheet owned by the user.

The extension may handle current job-page data, Google Sheet metadata and necessary sheet values, and the limited local settings required for the product to work.

This data is used only to extract job details, create and manage the user's own Google Sheet, check duplicate entries, and write records to the selected sheet.

The extension does not send job data to a developer-owned server, does not sell user data, and does not use the data for advertising, profiling, or resale.

Users may disconnect Google access, delete their own records from Google Sheets, or uninstall the extension at any time.
